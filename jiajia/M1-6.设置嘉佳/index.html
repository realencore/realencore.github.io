    M1-6.设置嘉佳¶

    本页面设置延时机制说明

    为避免暴力使用造成脆弱的 RBT 服务端崩溃，本页面功能设置不做实时保存，各功能开关设置均遵循 5 秒延时规则。

    设置提交规则：

    1. 本地每 5 秒检测一次功能开关的设置状态，有设置变更则保存提交至 RBT （即用户在 5 秒内开关「你好，嘉佳」功能一万次只向 RBT 发送一次设置请求）
    2. 用户在离开本页面时，检测设置状态，有设置变更则保存提交至 RBT

    功能开关设置定义：

    「你好，嘉佳」「自由对话模式」的开启关闭设置，「休息提醒」「夜间休眠」的开启关闭设置、时间设置。

    M1-6-1.设置嘉佳¶

    「你好，嘉佳」默认开启，其余功能默认关闭

    1.Robot X 语音唤起开关

    2.自由对话模式开关

    3.见 M1-6-2 A.休息提醒

    4.见 M1-6-2 B.夜间休眠

    5.跳转 M1-6-4.设备信息

    M1-6-2.防沉迷¶

    A.休息提醒 MOB 设置¶

    1.开启休息提醒功能时，显示提醒间隔选项 cell，显示字段：提醒间隔 提醒间隔时长 ，点击该 cell 弹出 ActionSheet 选择提醒间隔

    默认提醒间隔

    默认间隔为 每 30 分钟

    2.高亮当前所选项，可选择提醒间隔时长：

    1. 每 20 分钟
    2. 每 30 分钟
    3. 每 45 分钟
    4. 每 60 分钟

    操作说明：

    • 点击 时长 ，设置所选时长为提醒间隔，关闭 ActionSheet
    • 点击 取消 关闭 ActionSheet，不变更当前设置
    • RBT 接收到 MOB 新的提醒间隔设置时，释放旧的计时任务，计时器重新计时

    RBT 休息提醒流程¶

    1.用户点亮屏幕时休息提醒计时器开始计时，计时前提：MOB 端开启休息提醒功能

    2.到达休息触发点前10秒弹出倒计时提示，提示文案：x 秒后休息喽

    3.到达休息触发点时进入休息模式开始 5 分钟计时，并切换至 Robot X，TTS 提醒用户休息。TTS 过程不可打断，提醒语料见 RBT 休息提醒语料

    休息时间

    休息时间 5 分钟为固定值，用户无自定义休息时间权限

    时间轴示例，以每 30 分钟休息一次为例：

    4.在休息模式内功能限制见 C.休息提醒/夜间休眠功能限制

    5.在休息模式内点亮屏幕，Robot X 提示用户当前是休息时间，TTS 播放完后熄屏，提示语：baby_name，现在是休息时间哦，一会再玩吧。

    6.5 分钟后点亮屏幕显示 Launcher，从点亮屏幕时开始新一轮计时

    休息提醒计时规则

    1.计时器开始计时：点亮屏幕时
    2.在以下情况释放计时任务，取消计时：A.用户连续使用嘉佳，在未达到休息触发点 熄屏 5 分钟 或以上;B.RBT 使用视频/语音聊天 app 时

    如何退出休息提醒？

    1.5 分钟计时后自动退出
    2. MOB 设置嘉佳 -> 休息提醒 关闭休息提醒开关

    B.夜间休眠¶

    1&2.开启夜间休眠，显示休眠时间选项 cell ，默认开始于 21:30，结束于次日 7:00，，点击弹出 datePicker 选择开始、结束时间

    • 开启后，嘉佳在设定时间进入休眠模式
    • 若当前时间正处于休眠时间段，嘉佳进入休眠

    3.点击关闭 datePicker，不更改时间设置

    4.点击保存所选时间，关闭 datePicker

    5.datePicker 时间格式为 24 小时制

    时间设置规则：

    1. 开始时间 < 结束时间 ≤ 24 : 00， 则结束时间为当日内，结束时间显示设置的时间“xx : xx”
    2. 结束时间 < 开始时间，则结束时间为次日， 结束时间显示设置的时间“次日 xx : xx”；
    3. 设置开始时间与结束时间一样，或两者间隔少于15分 钟，toast 提示 休眠时间间隔不能少于15分钟，并将设置的对象恢复为原来设置的值。
    4. 保存用户设置的时间，用户关闭再次开启时，显示上次设定的休眠时间。

    RBT 夜间休眠流程¶

    1.正常运行

    2.到达休眠时间点前10秒弹出倒计时提示，提示文案：x 秒后休息喽

    3.到达休眠时间点时进入休眠模式，切换至 Robot X，TTS 提醒用户。TTS 过程不可打断，提醒语料见 RBT 夜间休眠提醒语料-进入休眠

    时间轴示例，以默认 21:00～次日 7:00 为例：

    4.休眠模式内功能限制见 C.休息提醒/夜间休眠功能限制

    在休眠模式点亮屏幕，显示 Robot X 并播放两遍打呼噜音效，3 秒后熄屏

    5.休眠结束，嘉佳屏幕亮起，音量恢复到休眠前水平，唤起 Robot X TTS 提示 。该过程不可打断，说完话后转到 Robot X模式，所有功能恢复。提醒语料见 RBT 夜间休眠提醒语料-结束休眠

    如何退出夜间休眠模式？

    1.在休眠结束时间自动退出
    2. MOB 设置嘉佳 -> 夜间休眠 关闭夜间休眠开关

    RBT 夜间休眠特殊情况¶

    1. 夜间休眠不受嘉佳开关机、重启的影响。重启时若检测到该时间正处于休眠时间，弹出机器人模式，转到 RBT 夜间休眠流程 3。检测到该时间不处于休眠时间，则正常运行无影响。
    2. 夜间休眠过程中，点亮屏幕，长按 Power 键，可以调出关机重启的弹窗。
    3. 关机重启弹窗交互遵循 RBT 全局规范

    C.休息提醒/夜间休眠功能限制¶

    功能名称 休息/休眠状态时是否可用
    Robot X 不支持语音对话、主动交互、多模态、捏手、戳脸、摇晃嘉佳
    Launcher 不可用
    RBT app 1.可通过弹窗进入设置界面
    2.可接听语音、视频来电
    3.可被远程监控
    硬件-电源键 可用，遵守全局规范
    硬件-返回键 1.Robot X 不可用
    2.设置界面/输入界面：返回 Robot X
    3.弹窗状态下：关闭弹窗
    硬件-Home 1.熄屏时：点亮屏幕
    2.亮屏时：不可用
    硬件-语音键 不可用
    硬件-音量键 不可用

    D.异常情况- RBT 网络断开连接¶

    1.休眠/休息模式

    2.RBT Wi-Fi 中断

    3.点亮屏幕弹出断网弹窗 R10-Alert-Network-Disconnect。弹窗不影响嘉佳播放完呼噜声后自动熄屏，若用户不关闭弹窗，熄屏后再次点亮屏幕，弹窗依然还在。

    4.点击设置按钮，进入输入密码界面 R5-1.进入设置的密码控制。当停留在输入密码界面、设置界面两个界面时，屏幕自动熄灭时间遵守全局规范，自动熄屏后再次点亮屏幕，还停留在该界面。

    5.密码正确，进入设置-WiFi设置界面 R5-2.网络设置。
